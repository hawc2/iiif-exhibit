# iiif-exhibit

A small Hugo site that showcases a curated set of Allied war posters from Temple University Libraries using IIIF image services.

## Run locally

```bash
hugo server -D --source /Users/alexwermer-colan/Code/Hawc2/iiif-exhibit
```

## Data

Poster records live in `data/posters.json`, with IIIF image service URLs derived from ContentDM manifest IDs.
